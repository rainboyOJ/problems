## 题目




### 题目描述

已知正整数 $n$ 是两个不同的质数的乘积，试求出两者中较大的那个质数。




### 输入格式
输入一个正整数 $n$。




### 输出格式

输出一个正整数 $p$，即较大的那个质数。




### 样例


输入/输出 # 1

::: line
```
21
```

```
7
```
:::





### 说明/提示
$1 \le n\le 2\times 10^9$

NOIP 2012 普及组 第一题
