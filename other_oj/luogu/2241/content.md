## 题目


### 题目背景 
1997年普及组第一题




### 题目描述

有一个 $n \times m$ 方格的棋盘，求其方格包含多少正方形、长方形（不包含正方形）。



### 输入格式
一行，两个正整数 $n,m$（$n \leq 5000,m \leq 5000$）。




### 输出格式

一行，两个正整数，分别表示方格包含多少正方形、长方形（不包含正方形）。



### 样例


输入/输出 # 1

::: line
```
2 3
```

```
8 10
```
:::
