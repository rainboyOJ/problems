## 题目




### 题目描述

我们知道计算机的大小单位有 B, KB, MB 等，现在请你编写一个计算器进行大小单位换算。

提示，$1$ GB = $2^{10}$ MB = $2^{20}$ KB = $2^{30}$ B。



### 输入格式
输入一个字符串，由一个 0 到 1023 之间的整数，一个单位名称（大写字母），一个等于号和问号，以及第二个单位名称。

单位只有可能是 `GB`,`MB`,`KB`,`B` 之间的一个。



### 输出格式

输出一个数字，精确到小数点后 6 位。



### 样例


输入/输出 # 1

::: line
```
128MB=?GB
```

```
0.125000
```
:::

输入/输出 # 2

::: line
```
64MB=?KB
```

```
65536.000000
```
:::
