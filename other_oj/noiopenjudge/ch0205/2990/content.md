## 描述


符号三角形的第1行有n个由“+”和”-“组成的符号 ，以后每行符号比上行少1个，2个同号下面是”+“，2个异号下面是”-“ 。计算有多少个不同的符号三角形，使其所含”+“ 和”-“ 的个数相同。n=7时的1个符号三角形如下:+ + - + - + + + - - - - +  - + + + -   - + + -    - + -     - -      +

## 输入


每行1个正整数n<=24,n=0退出.

## 输出


n和符号三角形的个数.

## 样例输入


```
15
16
19
20
0
```


## 样例输出


```
15 1896
16 5160
19 32757
20 59984
```
