## 描述


有一个箱子容量为 v（0<=v<=20000），同时有 n 个物品（1<=n<=30），每个物品有一个体积。
要求从 n 个物品中，任取若干个装入箱内，使箱子的剩余空间最小。请计算并输出最小的剩余空间。

## 输入


第一行是一个整数 v (0<=v<=20000) 表示箱子容量。第二行是一个整数 n (1<=n<=30) 表示物品数量。接下来 n 行，每行一个正整数（不超过10000），分别表示这 n 个物品各自的体积。

## 输出


一个整数，表示箱子最小剩余空间。

## 样例输入


```
24
6
8
3
12
7
9
7
```


## 样例输出


```
0
```
