## 描述


给定一组字符串，按指定的排序方式输出这些字符串。排序可是自然顺序（inc）、自然逆序（dec）、忽略大小写顺序（ncinc）、忽略大小写逆序（ncdec）等。

## 输入


输入有多行，第一行为一个表明排序方式的字符串见题面，第二行为字符串的数目。其余各行每行一个字符串，字符串中间可能空格，前后也可能有空格，但前后的空格要忽略。

## 输出


输出也有多行，按指定的顺序输出输入的字符串。

## 样例输入


```
ncdec
3
Hello World!
You're right!
haha! you're wrong!
```


## 样例输出


```
You're right!
Hello World!
haha! you're wrong!

```
