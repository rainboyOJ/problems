## 描述


Given a sequence of N number. Find the mode M in the sequence so that the number of M's appear in the sequence is maximum.

## 输入


First line: one positive integers N (N <= 100).Second line: N positive integers (<= 100000).

## 输出


One integer M.

## 样例输入


```
7
1 3 2 4 4 3 3

```


## 样例输出


```
3
```
