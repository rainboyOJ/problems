## 描述


Given a sequence S of N numbers. Find the least index id so that S[id] equals to K.

## 输入


First line: two positive integers N (N <= 100) and K (K <= 100000).Second line: N positive integers (<= 100000).

## 输出


One integer id.

## 样例输入


```
8 5
2 4 7 8 5 3 9 5
```


## 样例输出


```
5
```
