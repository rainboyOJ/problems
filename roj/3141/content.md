给定N个正整数$A\_1,A\_2,…,A\_N$，从中选出若干个数，使它们的和为M，求有多少种选择方案。

#### 输入格式

第一行包含两个整数N和M。

第二行包含N个整数，表示$A\_1,A\_2,…,A\_N$。

#### 输出格式

包含一个整数，表示可选方案数。

#### 数据范围

$1 \le N \le 100$,  
$1 \le M \le 10000$,  
$1 \le A\_i \le 1000$

#### 输入样例：

```
4 4
1 1 2 2
```

#### 输出样例：

```
3
```
