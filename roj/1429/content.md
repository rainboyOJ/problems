### 【题目描述】

在一个数轴上有$n$条线段，现选取其中$k$条线段使得这$k$条线段两两没有重合部分，问最大的$k$为多少?

### 【输入】

第一行为一个正整数$n$，下面n行每行$2$个数字$a\_i，b\_i$,描述每条线段。

### 【输出】

输出文件仅包括1个整数，为$k$的最大值。

### 【输入样例】

```
3
0 2
2 4
1 3
```

### 【输出样例】

```
2
```

### 【提示】

【数据规模】

对于20%的数据，$n≤10$。

对于50%的数据，$n≤1000$。

对于70%的数据，$n≤100000$。

对于20%的数据，$n≤1000000,0≤a\_i < b\_i≤1000000$。
