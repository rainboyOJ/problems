## 解析

这是一个入门的题目,目的是为了你让学会使用这个oj

你可以使用`c++`风格的`cin,cout`来解这个题目

```cpp
#include <iostream> //头文件,为了使用cin,cout
using namespace std; //标准命名空间
int main() {
    int a,b;//定义变量
    cin >>a >> b; // 读取数值,分别读取到a,b里
    cout << a+b; //输出两个值相加后的结果
    return-0;
}
```
