给定一个长度为 $n$ 的数列 ${a\_1,a\_2,…,a\_n}$，每次可以选择一个区间 \[l,r\]，使下标在这个区间内的数都加一或者都减一。

求至少需要多少次操作才能使数列中的所有数都一样，并求出在保证最少次数的前提下，最终得到的数列可能有多少种。

#### 输入格式

第一行输入正整数$n$。

接下来$n$行，每行输入一个整数，第i+1行的整数代表$a\_i$。

#### 输出格式

第一行输出最少操作次数。

第二行输出最终能得到多少种结果。

#### 数据范围

$0 < n \le 10^5$,  
$0 \le a\_i <2147483648$

#### 输入样例：

```
4
1
1
2
2
```

#### 输出样例：

```
1
2
```
