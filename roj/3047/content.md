给出一个表达式,其中运算符仅包含`+,-,*,/,^`（加 减 乘 整除 乘方）要求求出表达式的最终值。

数据可能会出现括号情况，还有可能出现多余括号情况。

数据保证不会出现大于或等于$2^{31}$的答案。

数据可能会出现负数情况。

#### 输入格式

输入仅一行，即为表达式。

#### 输出格式

输出仅一行，既为表达式算出的结果。

#### 输入样例：

```
(2+2)^(1+1)
```

#### 输出样例：

```
16
```
