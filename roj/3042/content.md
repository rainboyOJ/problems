在给定的N个整数$A\_1，A\_2……A\_N$中选出两个进行xor（异或）运算，得到的结果最大是多少？

#### 输入格式

第一行输入一个整数N。

第二行输入N个整数$A\_1$～$A\_N$。

#### 输出格式

输出一个整数表示答案。

#### 数据范围

$1 \le N \le 10^5$,  
$0 \le A\_i < 2^{31}$

#### 输入样例：

```
3
1 2 3
```

#### 输出样例：

```
3
```
