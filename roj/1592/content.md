### 【题目描述】

原题来自：SGU 223

在 $n×n$ 的棋盘上放 $k$ 个国王，国王可攻击相邻的 $8$ 个格子，求使它们无法互相攻击的方案总数。

### 【输入】

只有一行，包含两个整数 $n$ 和 $k$。

### 【输出】

每组数据一行为方案总数，若不能够放置则输出 $0$。

### 【输入样例】

```
3 2
```

### 【输出样例】

```
16
```

### 【提示】

样例输入 2

```
4 4
```

样例输出 2

```
79
```

数据范围与提示：

对于全部数据，$1≤n≤10,0≤k≤n^2$ 。
