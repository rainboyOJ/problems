### 【题目描述】

原题来自：SCOI 2009

Windy 定义了一种 Windy 数：不含前导零且相邻两个数字之差至少为 $2$ 的正整数被称为 Windy 数。

Windy 想知道，在 $A$ 和 $B$ 之间，包括 $A$ 和 $B$，总共有多少个 Windy 数？

### 【输入】

一行两个数，分别为$A,B$。

### 【输出】

输出一个整数，表示答案。

### 【输入样例】

```
1 10
```

### 【输出样例】

```
9
```

### 【提示】

样例输入 2

```
25 50
```

样例输出 2

```
20
```

数据范围与提示：

20% 的数据，满足 $1≤A≤B≤10^6$ ；

100% 的数据，满足 $1≤A≤B≤2×10^9$ 。
