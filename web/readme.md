
使用 [bootstrap5](https://getbootstrap.com/docs/5.2/getting-started/vite/)的官方教程来使用vite+vue3+vue-router.来创建一下problems的展示页面


使用的包

- vite
- vue3
- bootstrap5
- vue-router
- pouchdb
- markdown-it

## 页面

| 页面 | 作用|
|--------------- | --------------- |
| `/` | 题目列表   |
| `/problem/:id` |显示题目|
| `/sets/:id` : | 题目集|
