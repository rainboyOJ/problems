有n个集合，m个操作，操作分为三种：

“1 a b” ——合并a,b所在集合;  
“2 k” ——回到输入的第k次操作之后的状态;  
“3 a b” ——询问a,b是否属于同一集合，是则输出1否则输出0。

#### 输入格式

第一行为n，m。

接下来m行描述了每个操作，按照题目描述中所述的格式。

每个操作强制在线，需要对输入的 a,b,k 进行运算得到真实的 a,b,k 后再执行操作，运算方法为 x = x xor last\_ans，last\_ans表示上一个询问的答案，其初始值为0。

#### 输出格式

对于每个询问操作，输出一个结果，每个结果占一行。

#### 数据范围

$0 < n,m \le 2\*10^5$

#### 输入样例：

```
5 6
1 1 2
3 1 2
2 1
3 0 3
2 1
3 1 2
```

#### 输出样例：

```
1
0
1
```
