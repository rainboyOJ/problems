给定一个N行M列的01矩阵A，A\[i\]\[j\] 与 A\[k\]\[l\] 之间的曼哈顿距离定义为：

$$dist(A\[i\]\[j\],A\[k\]\[l\])=|i-k|+|j-l|$$

输出一个N行M列的整数矩阵B，其中：

$$ B\[i\]\[j\]=min\_{1≤x≤N,1≤y≤M,A\[x\]\[y\]=1}⁡{dist(A\[i\]\[j\],A\[x\]\[y\])}$$

#### 输入格式

第一行两个整数n,m。

接下来一个N行M列的01矩阵，数字之间没有空格。

#### 输出格式

一个N行M列的矩阵B，相邻两个整数之间用一个空格隔开。

#### 数据范围

$1 \le N,M \le 1000$

#### 输入样例：

```
3 4
0001
0011
0110
```

#### 输出样例：

```
3 2 1 0
2 1 0 0
1 0 0 1
```
