## [题目描述]

阿什尼铺满广场后,用剩余的石板雕刻了n组($n<20$)数字，现在阿什尼想把数字联接成一排，组成一个最大的多位整数。

例如：$n=3$ 时，$3$ 个整数 $13$，$312$，$343$ 联接成的最大整数为：$34331213$

又如：$n=4$ 时，$4$ 个整数 $7$，$13$，$4$，$246$ 联接成的最大整数为：$7424613$



## [输入格式]

 - 第一行$n$表示有$n$组数字
 - 第二行，$n$个正整数

## [输出格式]

连接成的最大数

## [输入样例]

```
3
13 312 343
```

## [输出样例]

```
34331213
```
