给定一个自然数N，要求把N拆分成若干个正整数相加的形式，参与加法运算的数可以重复。

**注意**:

-   拆分方案不考虑顺序；
-   至少拆分成2个数的和。

求拆分的方案数 mod 2147483648的结果。

#### 输入格式

一个自然数N。

#### 输出格式

输入一个整数，表示结果。

#### 数据范围

$1 \le N \le 4000$

#### 输入样例：

```
7
```

#### 输出样例：

```
14
```
